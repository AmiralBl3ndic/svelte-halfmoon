# Svelte Halfmoon

A Svelte component library for using the [Halfmoon](https://www.gethalfmoon.com/) CSS framework.

## Installation

```sh
# Choose one or the other
yarn add svelte-halfmoon # If you use yarn
# -----
npm install svelte-halfmoon # If you use npm
```
